## ✨ About Me

> **Hunting bugs before they hunt users 🐛**

I'm a **QA Engineer / SDET** who enjoys finding problems early and making sure products feel solid, smooth, and reliable.  
I work across **manual testing, Playwright automation, API testing, and geospatial web apps**, with a strong focus on quality and user experience.

- 🔍 I write test cases that actually matter
- ⚡ I automate with **Playwright** and **Selenium**
- 🌐 I test APIs with **Postman**
- 🗺️ I’ve worked on **geospatial applications**
- 🧠 I like asking: *What could go wrong here?*
- 🎯 Detail-oriented, quality-driven, and always curious
